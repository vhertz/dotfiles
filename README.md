# dotfiles

## Usage

Use some `make` commands.

|   `command`   |            description             |
| :------------ | :--------------------------------- |
| `make clean`  | Remove the symlinks.               |
| `make deploy` | Create symlink to the dotfiles.    |
| `make init`   | Initialize to deploy the dotfiles. |
| `make list`   | Show the dotfiles to deploy.       |
